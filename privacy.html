4. Netzwerkzugriffe und externe Dienste

Für bestimmte Funktionen kommuniziert die App mit externen Diensten. Es werden dabei keine personenbezogenen Daten übermittelt.

4.1 RSS-Feeds abrufen

Um die von dir hinzugefügten RSS-Feeds zu laden, werden HTTP-Anfragen an die jeweiligen Feed-URLs gestellt. Diese Anfragen enthalten lediglich die Feed-URL und technisch notwendige HTTP-Header (User-Agent, Accept). Es werden keine persönlichen Daten übermittelt.

Da WebViews keine direkten Cross-Origin-Anfragen erlauben, werden beim Abruf über die WebView-Komponente folgende CORS-Proxy-Dienste als Zwischenstufe genutzt:

  • corsproxy.io (https://corsproxy.io)
  • AllOrigins (https://api.allorigins.win)
  • CodeTabs (https://api.codetabs.com)

An diese Dienste wird ausschließlich die URL des jeweiligen RSS-Feeds übermittelt. Für den Hintergrund-Refresh (WorkManager) werden RSS-Feeds direkt abgerufen, ohne CORS-Proxy.

4.2 Feed-Icons (Favicons)

Für die Anzeige von Feed-Icons nutzt die App den Google Favicon Service:

https://www.google.com/s2/favicons?domain={domain}&sz=64

Dabei wird die Domain des jeweiligen Feeds an Google übermittelt. Es gilt die Datenschutzerklärung von Google.

4.3 Benachrichtigungen

Benachrichtigungen werden ausschließlich als lokale Benachrichtigungen auf deinem Gerät erzeugt. Es werden keine Push-Notification-Dienste (wie Firebase Cloud Messaging) verwendet.

4. Network access and external services

For certain features, the app communicates with external services. No personal data is transmitted in these requests.

4.1 Fetching RSS feeds

To load the RSS feeds you have added, the app makes HTTP requests to the respective feed URLs. These requests contain only the feed URL and technically necessary HTTP headers (User-Agent, Accept). No personal data is transmitted.

Because WebViews cannot make direct cross-origin requests, the following CORS proxy services are used as intermediaries:

  • corsproxy.io (https://corsproxy.io)
  • AllOrigins (https://api.allorigins.win)
  • CodeTabs (https://api.codetabs.com)

Only the URL of the respective RSS feed is transmitted. For background refresh (WorkManager), RSS feeds are fetched directly without a CORS proxy.

4.2 Feed icons (favicons)

To display feed icons, the app uses the Google Favicon Service:

https://www.google.com/s2/favicons?domain={domain}&sz=64

The domain of the respective feed is transmitted to Google. Google's Privacy Policy applies.

4.3 Notifications

Notifications are generated exclusively as local notifications on your device. No push notification services (such as Firebase Cloud Messaging) are used.
